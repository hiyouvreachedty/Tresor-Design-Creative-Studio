# Tresor Design Creative Studio

Personal portfolio and design studio site — built with **TypeScript + React + Vite**, showcasing web design, UI/UX, and front-end development work.

[![Live Site](https://img.shields.io/badge/Live-tresor--design--creative--studio.vercel.app-blue)](https://tresor-design-creative-studio.vercel.app)

---

## Tech Stack

- **Framework:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

---

```
│   ├── pages/                    # Page-level components

│   └── utils/                    # Utility/helper functions
├── index.html
├── package.json
├── tsconfig.json
└── vite.config.ts
```

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
